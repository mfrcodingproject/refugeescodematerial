## Introduction

We have already learned how JavaScript, HTML and CSS work. That means that you are able to create complex and advanced applications using the main frontend languages

Said that, the complexity of the websites has evolved rapidly in the past years. At this moment, most of the web development requires great amounts of code to offer advanced products with an extraordinary user experience. Doing it requires a massive amount of time and effort.

That's when frameworks came in. A framework is a tool developed in JavaScript that simplifies some of the most complex and repetitive tasks of the web development. Let's say creating a new element on the DOM and adding some text will require to create the element, create the text, then add the text to the new node and finally insert this node into the DOM. Using a framework that would cost 1 line of code.

Interesting, isn't it? During this course, you'll learn about VueJS, one of the main JS frameworks.

### Components

Components are pieces of code that can be reused many times around you app. Using a component framework, as this one, it is quite easy to reuse parts of your HTML, CSS and JavaScript.

The component system is an important concept in Vue, because it's an abstraction that allows us to build large-scale applications composed of small, self-contained, and often reusable components.
