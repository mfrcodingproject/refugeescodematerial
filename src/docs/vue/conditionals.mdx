## Introduction

Sometimes we want to show or hide content base on certain conditions. Imagine when a user clicks a button, a new menu appears or if the API returns that the user is not an admin we might want to hide a special menu.

In a web application this happens very often and depending on the data, and how this data changes (we will see Vue events later) we can decide which parts of the HTML are being displayed.

## Goals

## Contents

Conditional rendering is the ability to control whether template code is rendered. 

### Conditional (v-if)

Using this condition in your template, you will decide if some parts of your HTML are rendered in your template. When we say rendered, we mean removed.

### Display (v-show)

Using v-show we will also decide which parts of our HTML will not be displayed to the user, but in this case, the HTML will be rendered. Using this condition will hide the HTML only visually.

- üóíÔ∏è v-show: [https://v3.vuejs.org/guide/conditional.html#v-show](https://v3.vuejs.org/guide/conditional.html#v-show)

### When to use each

The key difference is that v-if **conditionally renders** elements and v-show **conditionally displays** elements.
